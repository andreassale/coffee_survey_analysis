# Coffee Survey Analysis

## Overview
This project analyzes coffee consumption habits using survey data.  
The goal is to identify patterns in preferences, demographics, and brewing methods through exploratory data analysis (EDA).

## Objectives
- Clean and preprocess raw survey data
- Analyze demographic distributions
- Identify popular coffee types and brewing methods
- Extract actionable insights from user behavior

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook / Google Colab

## Analysis Highlights

### Demographics
- Most respondents fall within the **25–34 age range**
- Data suggests a concentration of young adult consumers

### Coffee Preferences
- **Drip coffee** is the most preferred brewing method
- Lower adoption of **cold brew** and **iced coffee**

### Consumption Patterns
- Daily consumption is dominant among respondents
- Variation in preferences across demographic segments

## Data Cleaning Steps
- Handled missing values
- Standardized categorical variables
- Removed inconsistent or invalid entries
- Prepared dataset for analysis

## Key Insights
- Coffee consumption is strongly habitual
- Simpler brewing methods dominate user preferences
- Younger users show greater variability in choices
